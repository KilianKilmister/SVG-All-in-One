# 更新日志

[English](./CHANGELOG.md) | 简体中文

本文件记录该项目的重要变更。

## [0.1.0] - 2026-04-11

### 新增
- 在独立面板中提供 SVG 源码 + 预览的分栏工作流。
- 顶部工具栏操作：格式化、清理、压缩、保存、导出 PNG、导出多倍率 PNG。
- 预览区元素交互编辑：
  - 拖动移动
  - 左旋/右旋（15 度步进）
  - 放大/缩小（10% 步进）
  - 删除选中元素
- 预览编辑状态管理：
  - 撤销 / 恢复
  - 脏状态与关闭面板保存提醒
  - 与左侧源码编辑器实时同步
- 选中节点右键颜色工具：
  - 修改颜色
  - 提取颜色
- 选中节点属性侧边栏：
  - 精确编辑 / 新增 / 删除属性
- SVG 诊断能力：
  - 非法/不常见属性检查
  - 重复 `id` 检查
  - 基础可访问性检查
- SVG 智能补全（snippet 风格）：
  - 标签、属性、常见属性值建议

### 变更
- 画布缩放交互改为 `Ctrl/Cmd + 鼠标滚轮`。
- 移除右下角缩略预览图。
